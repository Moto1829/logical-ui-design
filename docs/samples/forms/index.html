会員登録フォームの比較

左は認知負荷が高く誤操作しやすい例、右は理解しやすく操作しやすい例です。

学習トップへ サンプル一覧へ
Bad

悪い例

ラベル不足、低コントラスト、情報の詰め込み、曖昧なCTAにより、入力の負荷が高い状態です。

入力内容に問題があります

  • placeholderがラベルを兼ねており、入力開始後に文脈が消える
  • 文字色と境界が弱く、どこが入力欄か瞬時に分かりにくい
  • 主要操作が強調されず、どのボタンを押すべきか迷う
  • エラー理由が曖昧で、修正方法が分からない
Good

良い例

ラベル、補助説明、グルーピング、明確なCTAにより、短時間で理解して操作できます。

基本情報

フルネームで入力してください

8文字以上で設定してください

任意項目

入力内容は自動保存されません。送信前に確認してください。

  • ラベルが常に見えるため、入力中でも意味を見失わない
  • fieldsetで情報が整理され、視線移動と理解の負荷が下がる
  • 主要ボタンが一目で分かり、行動の優先順位が明確である
  • 補助テキストが具体的で、入力の迷いを減らせる
