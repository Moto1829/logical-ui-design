モバイルでは届くことと迷わないことが重要

狭い画面では、情報量よりも主要操作の位置と入力中の文脈維持が使いやすさを左右します。

学習トップへ サンプル一覧へ
Bad

悪い例

主要操作が上部に遠く、ラベルも薄く、入力中に文脈が失われます。

プロフィール編集
  • 主要操作が親指で届きにくい位置にある
  • プレースホルダーだけで入力項目の意味が消える
  • キーボード表示後の操作導線が見えない
Good

良い例

常設ラベルと下部CTAで、入力中でも意味と次の行動が失われません。

プロフィール編集 入力中
変更を保存する
  • 主要操作が下部にあり、片手で届きやすい
  • ラベルが残るため入力中も意味が分かる
  • 最終操作が常に見えていて迷いにくい
