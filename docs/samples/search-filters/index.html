検索とフィルタの役割を分ける

検索とフィルタは似ていますが役割が違います。両者が混ざると、ユーザーはどこから絞り込めばよいか迷います。

良い例

検索、絞り込み、並び替えを分離し、現在の条件も明確に示しています。
