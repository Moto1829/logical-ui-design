レビューコメント

期限切れ案件と通常案件の視覚差が小さく、優先順位が判断しにくいです。期限切れを上段に分離すると、営業担当が最初に見るべき情報が明確になります。また「実行」では結果が読めないため「対応を開始する」など操作結果が分かる文言が適切です。

  • 観察事実、理由、改善案が一続きで書かれている
  • 画面の目的に照らして指摘できている
  • 次の修正にそのまま使える
